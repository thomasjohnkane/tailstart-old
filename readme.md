# Tailstart
Get started with Laravel 5.5, Tailwind CSS, Stimulus JS, and Turbolinks

# Installation
1. `git clone` this repo
2. `composer install && cp .env.example .env && php artisan key:generate`
3. `yarn install && yarn watch`
